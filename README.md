# Travel Buddy · 爸爸旅行英语

一个给英语零基础长辈使用的旅行英语小网页。它不要求系统学习，只帮用户在机场、酒店、餐厅等常见场景中快速找到一句能立刻开口的话。

## 功能

- 6 个常用旅行场景：机场、酒店、餐厅、打车、找厕所、求助
- 每个场景包含 6 句短而实用的英语
- 每句话都有中文意思、中文读法提示和使用说明
- 点击按钮即可用浏览器自带的 Web Speech API 播放英文发音
- 大字体、大按钮、宽松留白，并适配手机屏幕
- 不需要后端、账号或外部 API

## 如何运行

最简单的方法：直接双击 `index.html`，用浏览器打开。

推荐使用较新的 Chrome、Edge 或 Safari，以获得更稳定的英文发音效果。第一次点击“播放发音”时，浏览器可能需要短暂加载系统语音。

也可以在项目文件夹中启动任意静态文件服务器。例如电脑已经安装 Python 时：

```bash
python3 -m http.server 8000
```

然后在浏览器中打开：

```text
http://localhost:8000
```

## 文件说明

- `index.html`：页面结构
- `style.css`：界面样式与手机适配
- `script.js`：场景内容、页面交互与语音播放
- `README.md`：项目介绍与运行方法

## 使用提醒

发音功能依赖浏览器和设备自带的语音。不同手机或电脑上的声音可能略有区别；如果设备静音，也会听不到发音。
