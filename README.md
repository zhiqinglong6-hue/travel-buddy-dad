# Travel Buddy · 龙老板旅行英语

一个给英语零基础长辈使用的旅行英语小网页。它不要求系统学习，只帮用户在机场、酒店、餐厅等常见场景中快速找到一句能立刻开口的话。

## 功能

- 8 个常用旅行场景：机场、酒店、餐厅、打车、找厕所、求助、购物、买药
- 共 49 句短而实用的旅行英语
- 每句话都有中文意思、中文读法提示和使用说明
- 点击按钮即可播放随项目提供的本地英文音频
- 支持收藏常用句和全屏“给别人看”模式
- 大字体、大按钮、宽松留白，并适配手机屏幕
- 不需要后端、账号或外部 API

## 如何运行

最简单的方法：直接双击 `index.html`，用浏览器打开。

推荐使用较新的 Chrome、Edge 或 Safari。发音使用 `assets/audio/` 中的本地 WAV 文件，不依赖 Web Speech API；发布到 GitHub Pages 后可直接点击播放。

也可以在项目文件夹中启动任意静态文件服务器。例如电脑已经安装 Python 时：

```bash
python3 -m http.server 8000
```

然后在浏览器中打开：

```text
http://localhost:8000
```

## 文件说明

- `index.html`：页面结构
- `style.css`：界面样式与手机适配
- `script.js`：场景内容、页面交互与本地音频播放
- `assets/audio/`：每句英语对应的本地发音文件
- `README.md`：项目介绍与运行方法

## 使用提醒

如果手机听不到声音，请调高“媒体音量”，不是铃声音量。部分手机在静音或省电模式下也可能限制媒体播放，可换用最新版 Chrome 或 Edge 再试。
